# 📊 HR Analytics Dashboard (Power BI)

## 📌 Project Overview

This project analyzes employee data to identify key factors driving attrition and workforce trends. The dashboard provides actionable insights for HR teams to improve employee retention.

---

## 📁 Dataset

* Total Employees: 1470
* Key Features: Age, Salary, Job Role, Department, Attrition, Education

---

## 📊 Dashboard Features

* Employee Count, Attrition Count & Attrition Rate
* Attrition by Age, Salary Slab, Education, and Job Role
* Department-wise filtering (HR, R&D, Sales)
* Gender-based attrition analysis

---

## 🔍 Key Insights

* Highest attrition observed in age group **26–35**
* Employees with salary below **5K** show maximum attrition
* Sales roles have higher attrition compared to other departments
* Majority attrition occurs in early years of employment
